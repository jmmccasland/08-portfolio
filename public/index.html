My Projects

Heflin Construction

Built for general contractors, Heflin Construction, in Hendersonville, TN. I designed and developed this site as a portfolio for the company to showcase their craftsmanship. Its currently still in development as we wait for photos and content, however it was my favorite project I worked on prior to attending the Iron Yard and I can't wait to finish it after I graduate.

dogYears()

DogYears, appropriately spelled dogYears() refering to the function used in development, is a single page javascript web app I developed during prework for the IronYard. I spent several days writing JS but never actually connected any interaction to a user interface. Enter dogYears(), an interactive web app that allows to instantly convert your age in human years, to dog years!

Surf and Paddle

Surf and Paddle is an Iron Yard mock blog site for a surfing company that requires the development of a multi-column layout using flexbox. Its a fun project that requires a few neat tricks using position absolute and overlays. Its also the first project I've developed using SASS, which is awesome btw.
